# Create a new Expo project with create-expo-app:
## Usage for Bun, npm, pnpm, and Yarn
- `npm create expo`
- `bun create expo`
- `pnpm create expo`
- `yarn create expo`

## Output help information with all available options
- `npx create-expo --help`
